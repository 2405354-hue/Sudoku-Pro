# Sudoku Pro

A modern Sudoku game built using HTML, CSS, and JavaScript.

## Features
- Random Sudoku generation
- Unique-solution puzzles
- Easy, Medium, Hard modes
- Timer
- Best Time tracking
- 3-life system
- Row, column, and box highlighting
- Arrow-key navigation
- Responsive design
